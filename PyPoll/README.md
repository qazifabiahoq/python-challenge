PyPoll Election Analysis:

This Python project involves a script that analyzes election data from the "election_data.csv" file. The analysis encompasses several key calculations:

*Determining the total number of votes cast.
*Creating a complete list of candidates who received votes.
*Calculating the percentage of votes each candidate won.
*Determining the total number of votes each candidate won.
*Identifying the winner of the election based on popular vote.


This Python script serves as a practical example of analyzing election data using programming, making it a valuable tool for similar tasks.

How to Use:

To execute the PyPoll analysis, run the `main.py` script using Python. The script reads data from the "election_data.csv" file and produces the analysis results.
Files

* `main.py`: The primary Python script for performing the election analysis.
* Resources/election_data.csv`: The dataset used for the analysis.

Analysis Results:

The results of the election analysis are stored in the `Analysis` folder.
